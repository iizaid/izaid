# خطط تطوير نظام العملاء والطلبات

هذا المجلد يحتوي الخطة المنظمة لتطوير قسم العملاء والطلبات ومساحة عمل العميل في مشروع Thumbnail Viewer.

الهدف من هذه الخطة هو تحويل النظام من واجهة إدارية بسيطة وغير مكتملة إلى مساحة عمل احترافية تساعد على:

- فهم كل عميل بسرعة.
- حفظ تفضيلات العميل وأسلوبه.
- إدارة الطلبات بدون تشويش مالي.
- تجهيز وتسليم الصور المصغرة بشكل احترافي.
- تقليل الأزرار غير المفيدة أو غير العاملة.
- بناء تجربة عملية تناسب شغل تصميم thumbnails الحقيقي.

## مبدأ مهم

هذا المشروع هو منصة portfolio وجلب عملاء، وليس نظام دفع مالي. لذلك يجب إزالة أو تجاهل أي عناصر داخل الواجهة توحي بوجود معاملات مالية حقيقية مثل:

- مكتمل الدفع
- الفواتير
- تحصيل المدفوعات
- الديون
- أي منطق مالي داخل مساحة العميل

الدفع يتم خارج الموقع عبر WhatsApp أو Instagram.
